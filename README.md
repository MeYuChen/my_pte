# my_pte

用于提高 PTE WE 练习效率的离线网页工具。

## 使用

直接打开：

```text
index.html
```

或者启动本地静态服务器：

```bash
python3 -m http.server 8765
```

然后访问：

```text
http://127.0.0.1:8765/
```

## 内容

- 模板 5 分钟默写
- 文章论点练习
- 单篇考核
- 综合考核
- 最近 5 次滑动窗口熟练度统计
